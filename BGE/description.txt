105. БГЭ

ограничение времени на тест: 1 секунда
ограничение памяти на тест: 256 мегабайт
ввод: standard
вывод: standard

При решении задачи не используйте строковых или символьных переменных и циклов. Задачу необходимо решить используя ветвления, числовые типы данных и встроенные в язык операции по работе с ними.

В Берляндии после окончания школы каждый школьник сдаёт Берляндский государственный экзамен (БГЭ) по ряду предметов. Петя и его друзья собираются учиться на программистов, поэтому сдали экзамен по информатике. За экзамен каждый школьник получает балл, который выражается целым числом от 0 до 100. В Берляндии принято переводить баллы, полученные за экзамен, в привычные школьные оценки от 2 до 5. Перевод осуществляется по следующим правилам:

от 0 до 28 баллов за БГЭ — оценка 2
от 29 до 52 баллов за БГЭ — оценка 3
от 53 до 77 баллов за БГЭ — оценка 4
от 78 до 100 баллов за БГЭ — оценка 5

Требуется написать программу, которая бы переводила балл за БГЭ в оценку.

Входные данные
В единственной строке записано одно целое число x (0 ≤ x ≤ 100) — балл за БГЭ.

Выходные данные
В единственной строке находится одно целое число - оценка от 2 до 5.

Примеры
input  => 0 
output => 2

input  => 100
output => 5
