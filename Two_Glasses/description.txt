202. Два стакана

ограничение времени на тест: 1 секунда
ограничение памяти на тест: 256 мегабайт
ввод: standard
вывод: standard

При решении задачи не используйте строковых или символьных переменных и циклов. Задачу необходимо решить используя ветвления, числовые типы данных и встроенные в язык операции по работе с ними.

Есть два стакана. Каждый вмещает 150 миллилитров воды. В первом налито a миллилитров жидкости, а во втором — b миллилитров жидкости. Сколько надо долить жидкости в каждый из стаканов, чтобы суммарно в них оказалось c (a + b < c ≤ 300) миллилитров. Выведите два целых числа. Переливать жидкость через края стаканов нельзя, переливать жидкость из стакана в стакан тоже запрещено. Если существует несколько решений выведите то, в котором минимален объём доливаемой во второй стакан воды.


Входные данные
В единственной строке даны три целых числа a, b, c (0 ≤ a, b ≤ 150; 1 ≤ c ≤ 300).

Выходные данные
Выведите два целых числа — объём воды, который нужно долить в каждый из стаканов.

Примеры:
input  => 140 10 170
output => 10 10

input  => 10 10 21
output => 1 0
