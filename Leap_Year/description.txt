106. Високосный год

ограничение времени на тест: 1 секунда
ограничение памяти на тест: 256 мегабайт
ввод: standard
вывод: standard

При решении задачи не используйте строковых или символьных переменных и циклов. Задачу необходимо решить используя ветвления, числовые типы данных и встроенные в язык операции по работе с ними.

Год високосный, если для номера этого года верно хотя бы одно из утверждений:

- делится на 4, но при этом не делится на 100;
- делится на 400.

Выведите количество дней в феврале года y.

Входные данные
Первая строка содержит единственное целое число y (1 ≤ y ≤ 10000) — искомый год.

Выходные данные
Выведите единственное целое число — количество дней в феврале года y.

Примеры
input  => 2012
output => 29

input  => 2013
output => 28
