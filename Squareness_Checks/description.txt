401. Проверка квадратичности

ограничение времени на тест: 0.5 секунды
ограничение памяти на тест: 256 мегабайт
ввод: standard
вывод: standard

При решении этих задач на С++ используйте класс std::string, не увлекайтесь char* и массивами char.

Задана строка s, состоящая из строчных латинских букв. Проверьте, является ли она квадратом — то есть является ли она строкой четной длины, у которой первая половина строки совпадает со второй.

Входные данные
В единственной строке задана строка s, состоящая из строчных латинских букв. Длина строки не превышает 105 символов.

Выходные данные
Выведите строку "Yes" если строка является квадратом и "No" -- если нет.

Примеры
input  => abab
output => Yes

input  => xyz
output => No

input  => acdc
output => No
