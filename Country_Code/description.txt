101. Код города

ограничение времени на тест: 1 секунда
ограничение памяти на тест: 256 мегабайт
ввод: standard
вывод: standard

При решении задачи не используйте строковых или символьных переменных, ветвлений и циклов. Задачу необходимо решить, используя числовые типы данных и встроенные в язык операции по работе с ними.

У вас есть телефонный номер, состоящий из 9 цифр. По данному номеру Вам нужно определить код города. Кодом города является трехзначное число, образуемое второй, третьей и четвертой цифрами телефонного номера.

Входные данные
В единственной строке входных данных задано целое число a (110000000 ≤ a ≤ 999999999) — телефонный номер. Гарантируется, что вторая цифра телефонного номера не равна 0.

Выходные данные
Выведите искомый код в виде трехзначного числа.
